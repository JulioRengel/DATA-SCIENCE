# 📊 Aprendizados em Data Science com Python

Este repositório reúne meus estudos iniciais em **Pandas**, **NumPy** e **Matplotlib**, com foco em **análise de dados** e **visualização gráfica**.

## 📝 O que aprendi
- Atalhos básicos no Jupyter Notebook para manipular células.  
- Importar dados de diferentes formatos (CSV, Excel, TXT, HTML).  
- Explorar dados com `.head()`, `.tail()`, `.columns`, `.index`.  
- Filtrar informações com `loc` e ordenar com `sort_values`.  
- Criar novas colunas e salvar dados manipulados em outros formatos.  
- Usar `groupby` para calcular médias e somas agrupadas.  
- Trabalhar com arrays no NumPy e entender dimensões.  
- Criar gráficos simples e formatados com Matplotlib.
